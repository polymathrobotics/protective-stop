# Archive — point-in-time engineering records

Everything in this directory is a historical record kept for
traceability, migrated verbatim from the original development tree — do
NOT treat any of it as current documentation. Several files describe
failure modes that have since been root-caused and fixed (notably the
2026-05 lwIP "wedge" era on the earlier XIAO ESP32-S3 hardware:
task-stack overflows and an lwIP cross-thread bug, resolved in
v15.20–v15.26), and they reference bench addresses, version numbers,
endpoints, and file paths that no longer exist. The current, verified
documentation lives one level up in `docs/` (start with the top-level
`README.md`); superseded copies of promoted reports have been removed
from here to avoid duplicates.
